## 📘 개요 (Overview)

이 프로그램은 파이썬으로 제작된 **콘솔 기반 숫자야구 게임**입니다.  
- 기본 규칙: 3자리 숫자를 맞히는 스트라이크·볼·아웃 시스템  
- 시즌 모드 & 한국시리즈 모드 지원  
- 일시정지(pause/unpause), 구단 해체(end), 치트 코드 등 포함  
- **교육 및 학습용 예제**로 제작되었으며,  
  **비상업적 환경에서의 연구·학습용 사용만 허가**됩니다.
